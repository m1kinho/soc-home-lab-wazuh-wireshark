SOC Home Lab – Wazuh & Wireshark

Entry-Level UK SOC Analyst Portfolio

Lab Objectives

  • Practice SOC alert triage using Wazuh SIEM
  • Analyze endpoint and authentication logs
  • Validate alerts with packet-level network analysis (Wireshark)
  • Document incidents in a SOC-style format

SOC Investigation Workflow

  1. Alert received in Wazuh
  2. Initial triage and severity assessment
  3. Log and context review
  4. Network traffic validation (Wireshark)
  5. Classification (True / False Positive)
  6. Recommendation and documentation

Skills Demonstrated

  • SOC alert triage (True/False Positive classification)
  • Log analysis and correlation
  • Network traffic analysis
  • Incident documentation
  • Secure lab architecture and segmentation
