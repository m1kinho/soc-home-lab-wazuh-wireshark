# SOC Home Lab – Wazuh & Wireshark (UK SOC Analyst Portfolio)

This repository documents a fully isolated SOC home lab built to practice
security monitoring, alert triage, and network traffic analysis.

The lab is designed to reflect real-world SOC Analyst L1 duties
commonly expected in UK-based Security Operations Centres.

---

## Lab Objectives
- Practice SOC alert triage using Wazuh SIEM
- Analyze endpoint and authentication logs
- Validate alerts with packet-level network analysis (Wireshark)
- Document incidents in a SOC-style format

---

## Lab Environment
- Fully isolated VirtualBox lab
- No access to home LAN or Internet
- Controlled attack simulations only inside the lab

---

## Tools & Technologies
- Wazuh SIEM
- Wireshark / tcpdump
- Kali Linux
- Windows 10 / 11
- Ubuntu Server
- Metasploitable3
- VirtualBox

---

## Skills Demonstrated
- SOC alert triage (True/False Positive classification)
- Log analysis and correlation
- Network traffic analysis
- Incident documentation
- Secure lab architecture and segmentation

---

SOC Analyst Level 1 / Junior Security Analyst
