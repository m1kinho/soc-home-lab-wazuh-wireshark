## Wazuh SIEM Overview

Wazuh is used as the central SIEM solution for log collection,
analysis, and alerting.

### Roles
- Centralised log aggregation
- Rule-based alert detection
- Endpoint monitoring

---

## Connected Agents
- Windows 10 endpoint
- Kali Linux
- Metasploitable3

---

## Key Log Sources
- Windows Security Events (login activity)
- Linux authentication logs
- Process execution events
